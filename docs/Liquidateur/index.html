Liquidateur

  • Jeux vidéo
  • Unity

Liquidateurs est un prototype de simulation, où plusieurs agents interagissent entre eux. Ce projet a été créé quelques jours de travail. Dans cette simulation, il y a deux types d’agents : les liquidateurs qui ramènent des déchets toxiques vers une benne, et des médecins qui viennent aider un liquidateur lorsqu’il est à terre.

Gros plan sur les agents se déplaçant pour effectuer leurs tâches.
